JVM | JVM参数及配置建议

发表于 2015-01-02 | 分类于 Java ， JVM
如题...2018修正，基于JDK1.8

本文是对照Oracle JDK1.8官方文档，重新修正的常用JVM参数列表。原版文章已删除。
https://docs.oracle.com/javase/8/docs/technotes/tools/unix/java.html

JVM参数说明

参数名称 含义 示例 注释
-Xms 初始堆大小 -Xms8G 应用启动后，JVM堆会增大到-Xms，而后按需向OS请求内存。因该过耗时，建议设置-Xms=-Xmx
-Xmx 最大堆大小 -Xmx8G 建议设置-Xms=-Xmx
-Xmn 年轻代大小 -Xmn256m Java堆 = 老年代 + 新生代。建议年轻代大小设置为整个堆大小的3/8。这里年轻代，指eden + 2 survivor space。-Xmn设置的时候初始值和最大值，（如果想分别设置，可用-XX:NewSize 、 -XX:MaxNewSize，没实践过不建议…）
-XX:PermSize 永久代大小 JDK1.8中删除了永久代

名词：
方法区–Permanent Generation
元空间–Metaspace
元信息–Class metadata
本地内存–Native memory
interned Strings–字符串常量
